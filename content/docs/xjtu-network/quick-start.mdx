### 常见问题及处理方法

#### Portal 认证时提示账号密码错误

#### Portal 认证时提示其他未知错误

问题描述：在进行有线或无线 Portal 认证时时出现 `无法弹出登陆页面` 或 `出现缺少必要的参数即将重定向` 等问题的解决办法

解决方法：

1. 打开浏览器，在地址栏中手动输入认证网址：[http://p.xjtu.edu.cn](http://p.xjtu.edu.cn)**（注意必须是 http 而不是 https）**；
2. 若无法进入认证页面，则需要进一步检查 `DNS/IP` 地址是否配置错误（[查看 DNS 是否自动获取方法](http://ask.zol.com.cn/x/2495509.html)）：
   1. 若 `DNS/IP` 不为自动获取：尝试将 `DNS/IP` 改为自动获取；
   2. 若 `DNS/IP` 为自动获取：尝试手动断开 Wi-Fi 连接并重新打开 Wi-Fi 开关（若为有线网络，则需要重新插拔网线）；
   3. 若仍无法进入认证页面：尝试在网络设置中进行重置网络并重启计算机。
